Commands

  • help — lists commands
  • weather — prints the local weather
  • connect <url> — opens that site
  • new <file> — creates a file
  • run <path> — runs file with default app
  • set assistant name <name>
  • set user name <name>
  • print assistant name / print user name
  • search <query> / search youtube <query>
  • save <n> <text> / load <n>
  • remind <min> <sec> <text>
  • calc <a> add|sub|mul|div <b>
  • coin • roll <sides> (each 1–N has 1/N chance)
  • wiki <topic>
  • joke • fact • time
  • dir <folder> • password <len> • ip
  • translate <src> <dst> <text>
  • convert <from> <to> <value> (bin↔hex, °F↔°C, ft↔m, yd↔m, in↔cm, B↔KB↔MB↔GB↔TB)
