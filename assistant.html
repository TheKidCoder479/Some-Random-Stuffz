Download Assistant

This "AI" Assistant is a program made in python by me that can automate your daily tasks!

It can be used to automate tasks such as: reminders, conversion, math, making files, etc

In the program you can list the commands but i'll list them here right on this page:

help - lists commands

weather - prints the local weather

connect [site url] - opens a site with that url

new [file name] - makes a file with that name

run [file path/name] - executes that file with the default program

set assistant name [name] - sets the ai's name and saves it to line 1 in config.txt

set user name [name] - sets the users name and saves it to line 2 in config.txt

print user name - prints line 2 of config.txt

print assistant name - prints line 1 of config.txt

search [query] - searchs and opens a google query with that query

search youtube [query] - searchs and opens a youtube query with that query

save [variable number] [content] - saves the content of the variable at the line (variable number) in variables.txt

load [variable number] - prints the contents of the line entered in variables.txt

remind [minutes] [seconds] [text] - prints text onto the screen in a set amount of time

calc [1] add/sub/mul/div [2] - arithmetic

coin - 50% Chance of Heads 50% Chance of Tails

roll [sides] - rolls a dice with that many sides, so a dice with 10 sides would have a 10% of each number 1-10

wiki [topic] - searchs wikipedia for the topic

joke - tells a random joke

fact - tells a random fact

time - prints the time, format: "12:00 AM January 1st 2000"

dir [folder path] - lists files in a folder

password [length] - generates a random string at the selected length then prints it using a-z, A-Z, 1-9, 0, - and _

ip - print ip

translate [original lang] [target lang] [text] - translates text

convert [original format] [target format] [content] - convert binary to hexadecimal or vice versa, convert fahrenheit to celsius or vice versa, convert feet to meters or vice versa, convert yards to meters or vice versa, convert inches to centimeters to vice versa, convert between bytes, kilobytes, megabytes, gigabytes and terabytes

clear - clears all text on screen except for start text

history - displays the last 50 inputs by the user during the current session line by line
